Web hosting

El "Web hosting" es un servicio que proporciona herramientas para construir, publicar y administrar un sitio web y hacerlo accesible a visitantes de todo el mundo.

En nuestro caso hemos utilizado "github" como web hostig. GitHub Pages es un servicio de alojamiento de sitio estático que toma archivos HTML, CSS y JavaScript directamente desde un repositorio en GitHub, opcionalmente ejecuta los archivos a través de un proceso de compilación y publica un sitio web.
